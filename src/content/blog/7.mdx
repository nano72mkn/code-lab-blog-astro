## TL;DR

`tsconfig.json`の lib に`dom`を追加

## 現象

リポジトリの分割作業で出たエラー。

```
Typescript error: Cannot find name 'Response'
```

解決策は、`tsconfig.json`の lib に`dom`を追加するだけ
